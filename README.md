## О проекте

**Карьерная книга** — это корпоративная платформа для управления компетенциями сотрудников. Система автоматизирует процесс оценки навыков, формирования индивидуальных планов развития (ИПР) и подбора учебных материалов.

### Основные возможности

- **Единый источник требований к компетенциям** для каждой должности
- **Импорт матриц компетенций** из Excel с автоматической валидацией
- **Индивидуальные карьерные траектории** для сотрудников
- **Канбан-доска** для визуализации прогресса развития
- **Каталог учебных материалов** с привязкой к компетенциям
- **Аналитика компетенций** для руководителей подразделений
- **Настройка интерфейса** (светлая/темная/системная тема)

## Особенности

- **Ролевая модель:** Администратор, Руководитель, Сотрудник с разграничением прав доступа
- **Импорт данных:** Загрузка матриц компетенций из Excel с валидацией структуры
- **Drag & Drop:** Перетаскивание компетенций и материалов между блоками
- **Канбан-доска:** Визуальное отслеживание статуса развития компетенций
- **Адаптивный дизайн:** Поддержка мобильных устройств и различных разрешений экрана
- **Темы оформления:** Светлая, темная и системная темы с сохранением в localStorage

## Ролевая модель

### Администратор
- Полный доступ ко всем функциям системы
- Импорт матриц компетенций из Excel
- Управление компетенциями и учебными материалами
- Настройка ролей пользователей
- Управление системными параметрами

### Руководитель
- Просмотр и анализ компетенций сотрудников своего подразделения
- Назначение учебных материалов сотрудникам
- Фиксация оценок компетенций
- Просмотр статистики прогресса подчиненных
- Управление учебными материалами подразделения

### Сотрудник
- Просмотр своей карты компетенций
- Формирование индивидуального плана развития (ИПР)
- Просмотр требований к следующим позициям в отделе
- Доступ к каталогу учебных материалов
- Настройка уведомлений и интерфейса

## Установка и запуск

### Предварительные требования

- Node.js (рекомендуется версия 18.0.0 или выше)
- npm

### Клонирование репозитория

```bash
git clone

cd Frontend
```
### Установка зависимостей
```bash
npm install
```
### Запуск приложения в режиме разработки
```bash
npm start
```
### Сборка для продакшена
```bash
npm run build
```
